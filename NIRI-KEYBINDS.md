# Niri Keybinds (CachyOS Defaults)

Mod key = Super (Windows key)

## Application Control
| Keybind | Action |
|---------|--------|
| Super + Return | Open terminal |
| Super + Space | Open Wofi (launcher) |
| Super + B | Open Firefox |
| Super + E | Open file manager |
| Super + Q | Close focused window |
| Super + Alt + L | Lock screen |

## Workspace Navigation
| Keybind | Action |
|---------|--------|
| Super + 1-9 | Switch to workspace |
| Super + Tab | Previous workspace |
| Super + Scroll | Move between workspaces |

## Window Navigation
| Keybind | Action |
|---------|--------|
| Super + Arrow keys | Focus windows/columns |
| Super + H/J/K/L | Focus windows/columns (vim-style) |
| Super + Home | Focus first column |
| Super + End | Focus last column |

## Window Management
| Keybind | Action |
|---------|--------|
| Super + Ctrl + 1-9 | Move window to workspace |
| Super + Ctrl + Arrow keys | Move windows/columns |
| Super + Shift + Ctrl + Arrow keys | Move columns to monitor |

## Layout Controls
| Keybind | Action |
|---------|--------|
| Super + Ctrl + F | Expand column to available width |
| Super + C | Center column |
| Super + Equal | Increase column width |
| Super + Minus | Decrease column width |
| Super + Shift + Equal | Increase window height |
| Super + Shift + Minus | Decrease window height |

## Window Modes
| Keybind | Action |
|---------|--------|
| Super + T | Toggle floating mode |
| Super + F | Toggle fullscreen |
| Super + W | Toggle tabbed display |
| Super + O | Toggle overview mode |

## System Controls
| Keybind | Action |
|---------|--------|
| Super + Escape | Emergency escape (disable shortcut inhibitor) |
| Ctrl + Alt + Delete | Quit Niri |
| Ctrl + Shift + 1 | Screenshot |
| Ctrl + Shift + 2 | Screenshot (area) |
| Ctrl + Shift + 3 | Screenshot (window) |

## Media Keys
Volume and media keys work as expected.
